# 귀멸 서바이버 — Night Bloom 고도화 계획

## 플레이어 약속

좋아하는 귀살대원을 골라 각자의 호흡법으로 밤의 임무를 완수하고, 동이 틀 때까지 성장한 뒤 무잔을 직접 쓰러뜨리는 10분 내외의 고밀도 서바이버 액션을 만든다. 캐릭터 선택은 외형만 바꾸지 않는다. 시작 무기, 기술 실루엣, 이동 리듬, 무쌍, 성장 경로와 추천 임무가 모두 달라야 한다.

## 목표 감정과 핵심 동사

- 목표 감정: 빠르고 화려하지만 읽을 수 있는 전투, 팬이 알아보는 기술 판타지, 짧은 회복 뒤 다시 몰아치는 긴장감.
- 주 동사: 이동·대시·위치 선정.
- 보조 동사: 자동 호흡 공격의 궤적 유도, 젬 회수, 3택 성장, 무쌍 타이밍, 임무 경로 선택.
- 숙련 표현: 적 무리를 한쪽으로 모으고 투사체 축을 맞추기, 위험 임무를 먼저 끝내 보상 곡선을 당기기, 대시 무적과 무쌍을 보스 패턴에 맞추기.

## 코어 루프 계약

플레이어는 **이동·대시·무쌍 타이밍**으로 임무 지점과 적 무리를 정렬해 **밤의 연속 임무와 최종 토벌**을 달성한다. 포위·보스 패턴·시간 압박이 위험을 만들고, 처치와 임무 성공은 **경험치·호흡 강화·명기 보상·캐릭터 해금 진행도**를 준다. 체력이 0이면 빠르게 결과를 확인하고 재도전하며, 새벽 이후에도 무잔이 살아 있으면 승리하지 않는다.

## 캐릭터별 기술 아트 계약

생성 원화는 투명한 불투명 코어를 먼저 읽히게 하고, Three.js 메시·블룸·파티클은 짧은 보조층으로만 사용한다.

- 탄지로/기유: 물 용 파도와 나기 수막.
- 네즈코: 분홍 폭혈 발차기와 혈염 꽃잎.
- 미츠리: 분홍·연두 채찍검 리본.
- 시노부: 독침 코어와 나비 날개 잔상.
- 카나오: 꽃잎 초승달과 피안주안 집중선.
- 렌고쿠: 불꽃 호랑이와 화염 호흡의 굵은 궤적.
- 젠이츠: 금빛 직선 번개와 재진입 잔상.
- 이노스케: 톱날 이중 초승달.
- 무이치로: 시간차가 있는 3중 안개 참격.
- 우즈이: 쌍원호·폭약 구슬·박자 링.
- 사네미: 비취색 낫바람 회오리.
- 교메이: 철구·사슬·지면 파쇄.
- 히노카미/최종기: 검붉은 태양 화륜.

## 밸런스 계약

- 첫 의미 있는 선택: 8–15초.
- Lv10 도달: 150–210초.
- 자동 회귀 허용 범위는 초반 보스 20–65초, 무잔 25–90초로 두고, 대표 빌드의 체감 목표는 각각 25–45초와 40–70초로 맞춘다.
- 같은 역할 영웅의 60초 성과 편차: ±20% 이내.
- 동료 킬 기여: 총 25% 이하.
- 군중 무기는 보스 배율, 단일 대상 무기는 군중 배율을 데이터로 분리한다.
- 보스 히트는 공격 경로와 무관하게 한 피해 해결 정책을 사용한다.
- 처치 발동 효과는 `proc-from-proc` 차단과 동시 개수/내부 쿨다운을 가진다.
- 희귀 카드는 실제 가중치가 낮아야 하며, 표시만 희귀한 상태를 금지한다.

8064×4096 미사용 변형 시트는 로드하지 않는다. 반복 오브젝트는 공유/인스턴싱하고, 퀘스트·충돌 프록시는 시각 메시와 분리한다. 강한 VFX는 이벤트 기반·단수명으로 유지하며 모바일에서는 밝은 화면 점유율과 배너 중첩을 제한한다.

## 오늘 밤 실행 순서와 완료 증거

1. P0 무결성: 맵 전각 누적, 기술 배치 수명, 보스 중복 경고, 런 상태 초기화, 잘못된 승리 조건 수정.
2. 외부 아트: 캐릭터별 호흡 스프라이트를 직접 생성·크로마 제거·리사이즈·런타임 통합.
3. 전투: 명시적 기술 테마, 일관된 보스 피해, 과도한 중복타격/연쇄 발동 제한, 보스 타임라인 정렬.
4. 임무/맵: 바위 수련·등꽃 마을·무한열차·무한성·무잔전 계약과 모든 목표 방향표시.
5. UI/모바일: HUD 겹침, 44px 터치 타깃, 일시정지, blur/visibility 입력 해제, 배너 우선순위.
6. 성능: 거대 미사용 시트 제거, 모바일 파티클/PiP/후처리 예산 적용, 진단 정보 공개.
7. QA: 타입체크·프로덕션 빌드·콘솔 오류·데스크톱/모바일 활성 전투·실패/재시작·보스/퀘스트 상태를 검증한다.
8. 시각 게이트: 10개 점수 항목 모두 2 이상, 평균 2.3 이상을 목표로 새 스크린샷과 렌더러 수치로 재평가한다.

자동화가 아직 보호하지 못하는 경로는 완료로 숨기지 않고 잔여 위험에 기록한다.
